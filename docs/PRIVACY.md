# Privacidade e dados

## Dados processados

- Conteúdo de e-mails e anexos (para análise).
- UIDs do IMAP para controle de processamento.

## Armazenamento

- Estado mínimo em `phishbox-security/data/last_uid.json`.
- Não há persistência de conteúdo de e-mail fora do fluxo de processamento.

## Recomendações

- Use conta dedicada.
- Restrinja acesso ao host.
- Revise políticas internas antes do uso.
