# Segurança

## Boas práticas

- Utilize conta IMAP dedicada e senha de app.
- Não compartilhe arquivos `.env` ou `key.txt`.
- Restrinja acesso ao host que executa a stack.
- Mantenha Docker e imagens atualizados.
- Monitore logs de falhas e tentativas de login.

## Aviso

Este projeto é destinado a **triagem defensiva** de e-mails. Não use para qualquer atividade ilegal ou não autorizada.
